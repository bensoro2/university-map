สิ่งที่ได้เรียนรู้

การเรียนรู้เทคโนโลยีเว็บในห้องเรียน

ในการเรียนวิชานี้ ผมได้เรียนรู้เกี่ยวกับพื้นฐานของเทคโนโลยีเว็บ ซึ่งประกอบด้วย HTML, CSS และ JavaScript จากอาจารย์ผู้สอนที่มีความเชี่ยวชาญ การเรียนในห้องเรียนช่วยให้เข้าใจหลักการพื้นฐานและสามารถนำไปประยุกต์ใช้ได้จริง

  • ได้เรียนรู้การสร้างโครงสร้างเว็บไซต์ด้วย HTML5
  • เรียนรู้การจัดรูปแบบและการตกแต่งเว็บไซต์ด้วย CSS3
  • เข้าใจการใช้ JavaScript เพื่อเพิ่มความสามารถในการโต้ตอบกับผู้ใช้
  • เรียนรู้หลักการของ Responsive Web Design เพื่อให้เว็บไซต์แสดงผลได้ดีบนทุกอุปกรณ์

การเรียนรู้เพิ่มเติมด้วยตนเอง

นอกจากการเรียนในห้องเรียนแล้ว ผมยังได้ศึกษาเพิ่มเติมจากแหล่งการเรียนรู้ออนไลน์ต่างๆ เพื่อพัฒนาทักษะให้ดียิ่งขึ้น:

  • เรียนรู้ผ่านเว็บไซต์ W3Schools และ MDN Web Docs เพื่อศึกษาเทคนิคเพิ่มเติม
  • ดูวิดีโอสอนการเขียนโค้ดบน YouTube
  • ฝึกฝนทักษะการเขียนโค้ดบนแพลตฟอร์ม CodePen และ FreeCodeCamp
  • ศึกษาการใช้ไลบรารี Leaflet สำหรับการแสดงแผนที่บนเว็บไซต์

พื้นฐาน HTML ที่ได้เรียนรู้

HTML (Hypertext Markup Language) คือภาษามาร์กอัปที่ใช้สำหรับสร้างโครงสร้างเว็บเพจ ผมได้เรียนรู้หลักการสำคัญดังนี้:

  • การใช้ elements และ tags พื้นฐาน เช่น <div>, <p>, <h1>-<h6>, <ul>, <li>
  • การใช้ semantic elements เพื่อเพิ่มความหมายให้กับโครงสร้างเว็บไซต์ เช่น <header>, <nav>, <main>, <footer>
  • การใช้ attributes เพื่อเพิ่มคุณสมบัติให้กับ elements เช่น class, id, src, href
  • การสร้างฟอร์มและแบบฟอร์มต่างๆ ด้วย <form>, <input>, <select>
  • การใช้ <img> เพื่อแสดงรูปภาพและ <a> เพื่อสร้างลิงก์

พื้นฐาน CSS ที่ได้เรียนรู้

CSS (Cascading Style Sheets) ใช้สำหรับการจัดรูปแบบและตกแต่งเว็บไซต์ ผมได้เรียนรู้เกี่ยวกับ:

  • การเลือกใช้ selectors ต่างๆ เช่น element selector, class selector, id selector
  • การกำหนดสี, ขนาดตัวอักษร, และการจัดวางข้อความ
  • การใช้ box model (margin, border, padding, content) เพื่อจัดการขนาดและระยะห่าง
  • การใช้ display properties ต่างๆ เช่น block, inline, flex, grid
  • การสร้าง responsive design ด้วย media queries เพื่อให้เว็บไซต์แสดงผลได้บนทุกอุปกรณ์
  • การใช้ transitions และ animations เพื่อเพิ่มลูกเล่นให้กับเว็บไซต์

พื้นฐาน JavaScript ที่ได้เรียนรู้

JavaScript เป็นภาษาโปรแกรมมิ่งที่ใช้เพื่อเพิ่มความสามารถในการโต้ตอบกับผู้ใช้ ผมได้เรียนรู้:

  • พื้นฐานไวยากรณ์ของ JavaScript เช่น variables, operators, functions
  • การใช้ DOM (Document Object Model) เพื่อเข้าถึงและแก้ไขเนื้อหาในเว็บเพจ
  • การจัดการกับ events เช่น click, keydown, submit เพื่อตอบสนองต่อการกระทำของผู้ใช้
  • การใช้ fetch API เพื่อดึงข้อมูลจากแหล่งภายนอก
  • การเขียนฟังก์ชันง่ายๆ เช่น การเปิด/ปิดแท็บในหน้าเรียนรู้

การประยุกต์ใช้ในโครงงานนี้

ในการพัฒนาเว็บไซต์นี้ ผมได้นำความรู้ที่ได้เรียนมาประยุกต์ใช้ดังนี้:

  • ใช้ HTML ในการสร้างโครงสร้างหน้าเว็บไซต์ทั้ง 4 หน้า
  • ใช้ CSS ในการตกแต่งเว็บไซต์ให้ดูน่าสนใจและตอบสนองกับทุกขนาดหน้าจอ
  • ใช้ JavaScript ในการสร้างแท็บในหน้าสิ่งที่ได้เรียนรู้และในการทำงานของแผนที่
  • ประยุกต์ใช้ Leaflet API เพื่อแสดงแผนที่พร้อมข้อมูลสถานที่สำคัญ

การทำโครงงานนี้ทำให้ผมได้ฝึกฝนการนำทฤษฎีมาใช้ในทางปฏิบัติจริง และได้เรียนรู้วิธีการแก้ไขปัญหาที่เกิดขึ้นระหว่างการพัฒนา ซึ่งเป็นทักษะที่สำคัญมากในการพัฒนาเว็บไซต์

เป้าหมายในการเรียนรู้ต่อไป

ความรู้ที่ได้จากการเรียนในวิชานี้เป็นเพียงจุดเริ่มต้นของการเรียนรู้เทคโนโลยีเว็บ ในอนาคตผมตั้งใจจะศึกษาเพิ่มเติมในเรื่องต่อไปนี้:

  • เรียนรู้เฟรมเวิร์คต่างๆ เช่น React, Vue.js เพื่อพัฒนาเว็บแอปพลิเคชันที่ซับซ้อนมากขึ้น
  • ศึกษาเกี่ยวกับ Web APIs เพื่อเพิ่มความสามารถให้กับเว็บไซต์
  • เรียนรู้เกี่ยวกับ backend development เพื่อสร้างเว็บแอปพลิเคชันที่สมบูรณ์
  • พัฒนาทักษะด้าน UI/UX design เพื่อสร้างเว็บไซต์ที่ใช้งานง่ายและสวยงาม
